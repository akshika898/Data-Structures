We are given an array asteroids of integers.For each asteroid, the absolute value is its size, and the sign represents its direction
(pos meaning right, neg meaning left). Each asteroid moves at the same speed.
Find out the state of the asteroids after all collisions. If two asteroids meet, the smaller one will explode. 
If both are the same size, both will explode. Two asteroids moving in the same direction will never meet.

Input: asteroids = [5,10,-5]
Output: [5,10]
Explanation: The 10 and -5 collide resulting in 10.  The 5 and 10 never collide.

Input: asteroids = [8,-8]
Output: []
Explanation: The 8 and -8 collide exploding each other.


    public int[] asteroidCollision(int[] asteroid) {
        Stack<Integer> s=new Stack<Integer>();
        for(int i=0;i<asteroid.length;i++)
        {
            if(s.isEmpty() || asteroid[i]>0)
                s.push(asteroid[i]);
            else
            {
                while(true)
                {
                    int peek=s.peek();
                    if(peek<0)
                    {
                        s.push(asteroid[i]);
                        break;
                    }
                    else if(peek == -asteroid[i])
                    {
                        s.pop();
                        break;
                    }
                    else if(peek>-asteroid[i])
                        break;
                    else
                    {
                        s.pop();
                        if(s.isEmpty())
                        {
                            s.push(asteroid[i]);
                            break;
                        }
                    }                        
                }
            }
        }
        int a[]=new int[s.size()];
        for(int i=s.size()-1;i>=0;i--)
            a[i]=s.pop();
        return a;
    }
